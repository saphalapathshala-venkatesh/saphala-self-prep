PHASE 3 — IMPLEMENTATION (ONLY IF GAPS EXIST)
==================================================

If ANY gaps are found, implement fixes using the following rules:

----------------------------------------
1. TOTAL TIME FIX
----------------------------------------

Ensure everywhere:

total_exam_time = submittedAt - startedAt

Fallback:
- if startedAt missing → use Attempt.createdAt

----------------------------------------
2. FIRST-ATTEMPT FILTER
----------------------------------------

All analytics must use:

WHERE attemptNumber = 1

Applies to:
- rank
- leaderboard
- topper
- avg time
- heat maps

----------------------------------------
3. PER-QUESTION AVG TIME
----------------------------------------

For each question:

avg_time =
  AVG(timeSpentMs)
  FROM AttemptAnswer
  WHERE attemptNumber = 1
  AND attempted = true

Exclude:
- unanswered questions

----------------------------------------
4. HEAT MAP FORMULAS
----------------------------------------

Accuracy:
accuracy = correct / attempted * 100

Time ratio:
time_ratio = learner_time / avg_time

Risk classification:
- correct + fast → efficient
- correct + slow → time-heavy
- wrong + fast → rushed
- wrong + slow → time-heavy mistake

----------------------------------------
5. FOCUS SCORE
----------------------------------------

accuracy_risk = 100 - accuracy
unattempted_risk = unattempted %
time_risk = based on ratio buckets
wrong_density = wrong / total

Apply weighted formula exactly.
