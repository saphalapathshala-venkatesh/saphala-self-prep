Mobile field UX:
- Show Indian flag 🇮🇳

- Digits only, min=10, max=10

Validation:
- Email format
- Mobile length exactly 10
- Password min 8 chars
- Confirm password match
- Email and mobile must be unique (within in-memory store)

On success:
- Create user (in-memory)
- Create session
- Set HTTP-only cookie
- Redirect to /dashboard

B) LOGIN
- Page: /app/login/page.tsx (reuse existing UI)
- Fields:
  - Identifier (Email or Mobile)
  - Password

Logic:
- If identifier contains '@' → email
- Else → mobile
- Normalize input before lookup

On success:
- Create session
- Set cookie
- Redirect to /dashboard
